研究一手live2d的webgl sdk，为了用在网页和electron上。
e.g b站直播看板娘 这个插件

sdk

官网webgl免费，只要留邮就可下
链接
api文档完全什么都没有说明，只有实例+注释
下载的sdk里包含framework lib 和几个例子，因为chrome同源策略，要看换别的浏览器打开，edge可以， electron也没有这个问题。

关于simple的代码

  1. simple的代码非常简单，没有交互，看下就懂了，用webgl绘制
  2. 绘图循环中的requestAnimationFrame是浏览器的动画api，参数为 callback 和 element（可选）， element作用好像是为了该元素不在显示范围内时也保持动画，stackoverflow

simpleApp1解析

1. MatrixStack.js

WebGL is based off of OpenGL ES 2.0, so there is no built in matrix management or fixed function pipeline

实现了矩阵堆栈 (还好学过点opengl知道这个

2. ModelSettingJson.js

这个如其名，用于LAppModel

3.PlatformManager.js

抽象出的用于加载的工具类

3.Live2DFramework.js

实现了class L2DBaseModel

4.LAppModel.js

继承封装L2DBaseModel， LAppDefine是定义的全局变量，里面关于model的路径那些L2dBaseModel没有用到。

5.LAppLive2DManager

管理model，随机用model，路径定义在LAppDefine。有点击事件，hit头则随机表情，身体则随机motion

6.SampleApp1

主要是加了很多listener
