yilia在https下访问的问题

用hexo server测试时没问题，结果deploy后 文章列表和分享都没了

原因：
github page 在2016 6.15以后申请的都强制使用https
浏览器会报出混合错误

分享用的jiathis 好像只有http，我就不指望它了，反正也没啥用

archives的列表引用是iframe 莫名 /archives 使用了http开头，chrome会，edge不会
最后在main.js里找到iframe，把src改为完整url就可以了
