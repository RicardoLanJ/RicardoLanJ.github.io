关于python保存html后中文乱码

解决方法

with open("filename", "wb") as f
    f.write(file)
#即以二进制格式写入文件

猜的原因

py内部编码有 unicode 和 str
unicode 是 字符串
str 是 字节串
它们之间使用 encode decode来转换

可能windwos下 py保存默认为gbk编码 #记事本打开没问题 html因为有标明utf-8所以…
同时中间该字符串表示为unicode
因此很明显encode会出错

wb是写入二进制模式， 直接写入字节流， 就没问题
