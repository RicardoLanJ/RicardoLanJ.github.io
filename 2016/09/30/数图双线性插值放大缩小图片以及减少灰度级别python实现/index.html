数图像素级双线性插值放大缩小图片以及减少灰度级别-python实现

1.双线性插值的方法网上blog都有

主要就是把新图的像素点的值映射回原图的位置，落在原图的网格中，使用网格的四个顶点带入公式 ：
f(x,y) = f(i+u,j+v) = (1-u)(1-v)f(i,j) + (1-u)vf(i,j+1) + u(1-v)f(i+1,j) + uvf(i+1,j+1)

使用image库读图， numpy转换为二维矩阵，遍历像素点映射到原图套用公式即可。

2.灰度级别减少
设转换后级别为level, 则分为level段， 每段长seg = round(256 / (level - 1))
