haskell的几个练习

发表于 2016-10-24

其实是总结…

####1
QuickCheck要update cabel才能装

下面的句子可以执行所有quickcheck
{-# LANGUAGE TemplateHaskell #-}
return [] – need this for GHC 7.8
– quickCheckAll generates test cases for all ‘prop_*’ properties
main = $(quickCheckAll)

qucikcheck里 ==> 可以用于限制条件

infix infixl infixr用于设置运算符优先级

####2
haskell里没有循环，都是利用尾递归或者列表生成式，所以在进行迭代时如果次数太多内存会爆炸。尝试过各种方法，都会这样。找了很久发现一个seq函数， 函数原型是a -> b -> b，它的作用是立即计算a然后返回b。这就可以实现循环执行同一个操作，如下的牛顿迭代，可以观察发现内存不会无限增长

squareroot :: Float -> Float -> Integer -> Float
squareroot _ x0 0 = x0
squareroot r x0 n =  x `seq` squareroot r x (n-1)
    where x = dd r x0

####3
haskell 的\$和.运算符区别在于， \$用于去掉括号，在连续的使用函数时，f (g x) = f $ g x，而.更主要的目的是函数的链式调用， 可以把几个函数组合成一个函数，ex：

func = map (+ 1) . reverse
func [1,2,3]

map (+ 1) . reverse $ [1,2,3]

map (+ 1) $ reverse  [1,2,3]

他们的效果都一样
